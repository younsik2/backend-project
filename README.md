## Features

**User**

- 회원가입
  사용자 정보를 받아 새로운 사용자를 생성합니다.
- 로그인
  사용자 정보를 받아 사용자 유효성을 확인하고 토큰을 반환합니다.
- 장바구니
  토큰을 받아 사용자의 장바구니 정보를 불러옵니다.
  제품 추가, 삭제, 장바구니 전체 삭제가 가능합니다..

**Product**

- 제품의 타입, 카테고리, 제품 id를 받아 제품 정보를 반환합니다.

**Search**

- 검색
  검색어를 받아 해당 검색어를 포함하는 제품 목록을 반환합니다.

**Store**

- 점포 조회
  전체 매장의 정보를 반환합니다.
